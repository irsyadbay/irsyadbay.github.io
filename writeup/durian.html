Exposed .env File pada Sistem SIHARWAT

Irsyad Bayu Kurniawan — Web Security Researcher

Target

https://siharwatbbm.restuban.jatim.polri.go.id/.env

Deskripsi Sistem

Sistem ini berada pada subdomain resmi polri.go.id, yang menunjukkan bahwa aplikasi ini merupakan bagian dari infrastruktur internal Kepolisian Republik Indonesia, khususnya wilayah Jawa Timur.

Aplikasi SIHARWAT diduga merupakan sistem internal yang digunakan untuk kebutuhan operasional atau monitoring di lingkungan instansi. Sistem ini seharusnya hanya dapat diakses oleh pihak internal dan tidak diekspos ke publik.

Awal Penemuan

Saat melakukan pengamatan terhadap struktur server, ditemukan bahwa file .env dapat diakses secara publik tanpa autentikasi.

File ini biasanya berisi konfigurasi sensitif seperti kredensial database, API key, serta pengaturan layanan aplikasi.

Analisis Temuan

Akses terbuka terhadap file .env merupakan kesalahan konfigurasi serius pada server web. Hal ini dapat menyebabkan kebocoran informasi sensitif yang seharusnya tidak tersedia secara publik.

Dampak Potensial

  • Kebocoran kredensial database
  • Akses tidak sah ke sistem backend
  • Kompromi API key atau service internal
  • Potensi eskalasi ke sistem lain dalam jaringan internal

Rekomendasi

  • Blok akses publik terhadap file .env
  • Konfigurasi web server (Apache/Nginx) untuk deny access
  • Rotasi semua kredensial yang mungkin terdampak
  • Pastikan file konfigurasi tidak berada di direktori publik

Penutup

Temuan ini menunjukkan bahwa kesalahan konfigurasi kecil dapat berdampak besar terhadap keamanan sistem. Akses publik terhadap file konfigurasi seperti .env merupakan risiko serius yang harus segera ditangani.

Laporan ini disusun sebagai bagian dari praktik responsible disclosure untuk meningkatkan keamanan sistem.
